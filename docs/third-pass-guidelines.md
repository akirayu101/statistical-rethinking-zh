# 第三次精译规范

本轮不再把“贴近英文词序”当作准确。目标是在不改变数学含义、因果范围和作者语气的前提下，让中文读者第一次读到句子就能明白它在说什么。

## 三条复核轨道

1. **语义准确**：核对主语、条件、否定、比较对象、因果方向和不确定性范围。公式、变量、代码接口及原书勘误不因润色而改变。
2. **中文自然**：重组英语长句，补出中文所需的承接关系；删去无信息的名词化、被动式和重复词，但保留作者的幽默、比喻与警告强度。
3. **术语按语境**：专业概念保持全书一致，同时允许为读者理解补足必要信息。术语第一次出现、图轴、图例和无上下文标签优先使用完整名称；上下文已经明确的固定搭配可以保持简洁。

## 概率密度的处理

- 独立出现的 `density`，尤其是图轴、图例、替代文本和第一次解释，译为“概率密度”。
- 连续变量使用“概率密度”，离散变量使用“概率质量”或“概率”。不能把概率密度误说成某一点的概率。
- “后验密度”“核密度估计”“最高后验密度区间”“对数逐点预测密度”等固定术语，不机械扩写为冗长表达。
- 只有上下文明确讨论物理密度、种群密度等其他概念时，才使用相应领域含义。

## 公式、图与代码

- 润色公式前后的解释，不改公式本身的符号、上下标、条件竖线、括号层级和编号。
- 图内短标签必须脱离正文也能理解；正文图注要说明视觉编码与统计含义，而不只是复述左右位置。
- R、Stan、包名、函数名、参数名和可运行代码保持原样；中文代码注释、清单标题和输出解释使用同一套术语。
- 英文全称只在首次定义、官方缩写展开、作品标题或专门讨论英语词义时保留，并同时给出中文解释。

## 验收

第三轮账本以 PDF 页码为范围，分别记录语义准确、中文自然和语境化术语三条轨道。`scripts/audit_third_pass.py --require-complete` 还会检查可见中英混排、术语空格、独立图轴“密度”、术语表映射和图 10.2 的关键读者标签。
